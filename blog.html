Steam Next Fest is here!

February 26, 2025

Howdy folks, Will here,

It’s been a while since the last post, getting ready for Steam Next Fest, has been quite an adventure. Coming to learn how much is needed for a major event like this has definitely been a learning challenge.

From our end, getting the game prepared is only the beginning, we now have to make sure we stay streaming and ready to play with other players, a multiplayer game like ours is tough to entice new players if there aren’t any servers ready to play!

I’ll be sure to make updates (and maybe some statistics from this) as we move forward during the next couple weeks, but for now, that’s all from me!
